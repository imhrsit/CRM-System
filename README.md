# Mini CRM System

- `frontend/` — React app for UI
- `backend/` — Node/Express API server

- Frontend: React.js
- Backend: Node.js/Express.js
- Database: PostgreSQL
- AI Integration: OpenAI
- CI/CD: GitHub Actions
